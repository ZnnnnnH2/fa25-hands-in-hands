#### `<algorithm>`：

- `sort(begin, end)`
- `reverse(begin, end)` 反转
- `max_element(begin, end)` 找最大元素
- `min_element(begin, end)` 找最小元素
- `accumulate(begin, end, 0)` 求和（记得要 `#include <numeric>`）
- `count(begin, end, x)` 统计出现次数
- `find(begin, end, x)` 找到返回迭代器，找不到返回 end
- `unique` + `erase` 去重（高级一点可以只演示）

#### `vector` / `string` 常用方法

- `v.push_back(x)`、`v.size()`、`v.clear()`
- `s.length()` / `s.size()`
- `s.substr(pos, len)` 获取子串
- `s.find("abc")` 查找子串位置
- `stoi` / `to_string` 字符串 ↔ 整数（C++11 及以上）

- 计数、统计频率的神器：

  unordered_map<string, int> cnt;
  cnt[s]++;  // 统计字符串 s 出现次数
